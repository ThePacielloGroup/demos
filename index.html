The Paciello Group HTML Demos

This is a collection of demos of how to make standard HTML (i.e. non-interactive) idioms accessible. It is all clean HTML with no frameworks used. There is an Atom Feed and the articles are tagged and feature links to relevant parts of the W3C's WCAG documentation.

For more information, please visit the GitHub repository.
